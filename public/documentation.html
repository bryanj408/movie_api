INSTRUCTIONS:

My movie app will show a collection of movies that you can sort through and

learn about different movies, directors, actors, genres, etc.

You can create an account, login, and add movies to your favorites in your profile

ENDPOINT DOCUMENTATION:

Request URL HTTP Response Response
Return a list of all movies to the user /movies GET a JSON object holding data about all movies from server
Return data (description, genre, director, and image URL of a single movie by title to user /movies/title GET a JSON object containg data described above about a single movie
Return data about a genre (descritpion) by name/title (e.g. "drama") /movies/title/genre GET a JSON object holding data about a genre
Return data about a director (bio, birth year, death year) by name /movies/title/director GET a JSON object holding data about a director
Allow new users to register an account /users/id POST A response saying the info was accepted
Allow users to update their existing info /users/id PUT A response saying the user's info was update succesfully/failed
Allow users to add a movie to their list of favorites under their profile /users/id/favoriteMovie POST A response confirming that the movie was added
Allow users to remove a movie from their favorites list /users/id/favoriteMovie DELETE A response confirming that the nmovie was removed
Allow existing users to delete/deregister their existing account /users/id DELETE A response confirming that the account has been deleted/removed
